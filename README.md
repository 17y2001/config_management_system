# config_management_system

To run the project you can also do this : - 

**In VSCode terminal** - 
run this command : python -m venv env

**Install required packages** 
In the terminal, run: 

pip install fastapi sqlalchemy pydantic psycopg2-binary uvicorn

**Requirements-** 
fastapi
sqlalchemy
pydantic
psycopg2-binary
uvicorn
python-dotenv
